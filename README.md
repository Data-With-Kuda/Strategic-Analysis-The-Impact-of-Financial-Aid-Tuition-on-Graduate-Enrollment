# Strategic Analysis: The Impact of Financial Aid & Tuition on Graduate Enrollment

This repository presents a strategic analysis, developed as a **school project using data collected from the University of Louisville (UofL) institutional repository**.
The focus is on investigating the core drivers of volatility in graduate student enrollment trends, specifically examining the relationship between financial aid allocations and rising tuition costs.
